# 特性

核心内置模块

* ASM 内置支持

艺术性

* bootstrap - 自举（自己编译自己）
* 无关键词  - 无任何的自然语言关键词或保留字
* 导出导入  - 模块化
* 保持微型  - 大小真的很影响
* 保持优雅  - 不优雅又有何意义?
* FFI/DL    - 动态库及外部函数

可扩展内置模块

* JSON/BSON 
* 可扩充的语言模块，如正则式等.

未来

* 强化多线程、多进程、协程(multi-thread/multi-process/coroutine)
* 虚拟机及与平台无关的已编译中间二进制码
